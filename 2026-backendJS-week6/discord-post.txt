# 蛤？AI 家教跟網頁一問一答，哪有差
```
到這裡你可能會說：「疑？那跟平常在網頁問 AI，有什麼不同？」

差很多！真的要跑越來越複雜的程式時，最大的問題通常不是邏輯本身，而是小細節跟環境。
在本地端架 AI Agent 當家教，它能直接讀你的檔案、跑測試、幫你確認輸出——省掉一堆環境 error 來回貼 code 的時間。
~~（說真的也是因為我平常工作就在搞這個，實在不想再額外花時間踩坑了）~~
```

# How to Start？
很簡單，進到指定資料夾（這次就是第六週作業資料夾），然後輸入：
```
claude
```
再下這個 prompt：
```
現在這下面的 homework.js，我要完成它，教我一步步完成，最好是能在這邊互動式，我打對了後你就幫我填入，打錯了就再問我直到我打對。
```
